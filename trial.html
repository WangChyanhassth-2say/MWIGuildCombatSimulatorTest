导出当前安排 JSON

根据当前两个 Boss 阵容生成当前安排 JSON；无法识别角色 ID 的条目会列在下面。若已读取过 API，会直接复用当前页面内存缓存。

从 API 导入配装

API 地址只用于本次读取，不会写入页面缓存、完整预设导出或 localStorage。读取成功后会清空输入框；API 返回内容会暂存在当前页面内存中供导出当前安排 JSON 使用。
